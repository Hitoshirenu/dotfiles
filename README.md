# Dotfiles

**My dotfiles from Manjaro 17.1.10 i3 edition**

## My Setup

 * i3-gaps  -- wm
 * compton  -- compositor
 * polybar  -- statusbar
 * dunst    -- notification
 * nitrogen -- wallpaper
 * termite  -- Terminal
 * rofi     -- app launcher
 * ncmpcpp  -- music player (cmus previously)
